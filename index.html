Project 7: Ansible Jenkins Integration

Complete DevOps CI/CD Free-Style Project

created By Pratik Pardeshi

Project Overview

Developer write a code and push the code in github. Github is integrate with Jenkins. Jenkins job run compiling & building. after compiling jenkins its created a file and jenkins pushed this file in Ansible. Jenkis triggered a playbook written in Ansible. In this playbook I write a code for deploy a software on Web-server. so, Developer do any changes in code it directly change in web-server.

Project Steps

  • Create and configure 4 EC2 instances: Jenkins-Server, Ansible-Server, Web-Server, and Developer.
  • Setup Jenkins, including Java and plugin installations.
  • Install and configure Ansible for managing deployments.
  • Setup SSH key-based authentication between Jenkins, Ansible, and Web servers.
  • Create an Ansible playbook to deploy files to the Web-Server.
  • Integrate GitHub with Jenkins for automated build and deployment.
  • Verify deployment on the Web-Server and test code changes from the Developer system.
